Smart Wi-Fi Enabled Timetable and Automatic Bell System

A smart digital timetable and automatic bell system was designed and developed using an Arduino, ESP8266 Wi-Fi module, and P10 LED display modules. The system dynamically displays the current class, next class, date, day, and real-time clock information in an organized scrolling format.

The timetable data is stored in JSON format on GitHub, allowing the schedule to be updated remotely without modifying the hardware or reprogramming the microcontroller. Once connected to Wi-Fi, the ESP8266 automatically fetches the latest timetable data from the online repository and updates the display in real time.

An automatic motor-based bell mechanism is integrated into the system to ring at predefined class transition times. The bell timing logic is synchronized with the timetable, enabling fully automated operation without manual intervention. The system continuously checks the current time and triggers the bell before or at the end of each class period according to the uploaded schedule.

The P10 LED display provides clear long-distance visibility, making the system suitable for schools, colleges, coaching centers, and institutional environments. The project demonstrates the integration of IoT, real-time scheduling, wireless data communication, and embedded system automation into a centralized smart classroom management solution.

Key Features
Real-time display of:
Current class
Next class
Day and date
Current time
Automatic Wi-Fi synchronization
Remote timetable updates through JSON data hosted on GitHub
Automatic motor-based bell control
Fully automated class schedule management
P10 LED scrolling display for high visibility
Low-cost and scalable embedded system design
Technologies and Components Used
Arduino UNO / Mega
ESP8266 Wi-Fi Module
P10 LED Display
DC Motor Bell Mechanism
JSON Data Handling
GitHub Cloud Storage
Embedded C / Arduino IDE
Real-Time Clock Synchronization via Internet (NTP)
